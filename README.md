# GENERATE_RANDOM_IFT_USER_12M_4Q_BASE

## Формулировка задачи и техническое задание

Проект представляет собой базовую заготовку для генерации случайных пользователей IFT (If-Then) с параметрами 12M (12 месяцев) и 4Q (4 квартала).

**Техническое задание:**
- Создание базовой структуры проекта на Python
- Настройка системы логирования с уровнями INFO и DEBUG
- Подготовка инфраструктуры для дальнейшей разработки функционала генерации пользователей

## Подробное описание решения

Проект организован согласно стандартным правилам разработки:
- Основной код размещен в папке `src`
- Тестовые файлы хранятся в `src/Tests`
- Дополнительная документация в папке `Docs`
- Логи сохраняются в папку `log` с форматом: `Уровень_(тема)_годмесяцдень_час.log`
- Конфигурация и секретные данные в файле `.env`
